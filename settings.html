Name Generator
Use /generateName to test name generation.
If the local library is missing, the extension auto-loads fantastical from a CDN.
Supported kinds include: human, elf, dwarf, goblin, orc, ogre, halfling, gnome, fairy, highelf, woodelf, darkelf, halfelf, tavern, guild, mysticOrder, militaryUnit, adventure, and settlements: village, town, city, hamlet.
